Формат вывода:
    Обычный
    Сначала выводится header, после, через визуальный отступ в виде пустой строки выводится "Hello World"


Задание:
    Создаётся файл с неизвестным название, в котором есть заветное слово "Hello World".
    Далее в коде название этого файла придётся узнать.

    Создаётся JSON файл, в котором будет 1 root пользователь и 2 обычных.
    В root будут переписаны данные из файла в формате словаря, в виде "ключ:значение",
    где ключ - номер буквы, а значение - буква под этим номером.
